# Mes premiers programmes informatiques !
Quand j’étais petit (entre 8 et 11 ans), j’aimais beaucoup écrire des programmes en Basic (Visual Basic 4 sur Windows 95/98 et QuickBasic 4.5 sur DOS).

## Pour récupérer ces programmes, j’ai dû :
- Les copier sur disquettes depuis un ordinateur 386 (DOS / Windows 3.0) vers un ordinateur Pentium (Windows 95)
- Connecter l’ordinateur Pentium en Ethernet à mon routeur Internet
- Connecter mon ordinateur portable (Windows 11) au même routeur en Wi‑Fi
- Installer un serveur FTP sur l’ordinateur Windows 11 (seul protocole qui semble bien fonctionner entre Windows 98 et Windows 11)

## Ces programmes sont écrits de façon naïve, avec plein de spaghetti code et d’anti‑patterns !
Ce qui était très motivant avec l’informatique personnelle dans les années 90, c’était aussi cette forme de simplicité : le fait de programmer pour un usage individuel et de s’amuser sur son propre ordinateur.

# My First Computer Programs!

## When I was a kid (between 8 and 11 years old), I loved writing programs in Basic (Visual Basic 4 on Windows 95/98 and QuickBasic 4.5 on DOS).
To recover these programs, I had to:
- Copy them onto floppy disks from a 386 computer (DOS / Windows 3.0) to a Pentium computer (Windows 95)
- Connect the Pentium computer to my Internet router via Ethernet
- Connect my laptop (Windows 11) to the same router via Wi‑Fi
- Install an FTP server on the Windows 11 machine (the only protocol that seems to work reliably between Windows 98 and Windows 11)

## These programs are written in a naïve way, full of spaghetti code and anti‑patterns!
What was so motivating about personal computing in the 90s was also this sense of simplicity: programming for your own individual use and having fun on your own machine.
